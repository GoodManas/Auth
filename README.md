# Мое Python приложение

Это графическое приложение на Python, созданное с использованием PyQt6 и подключением к базе данных.

## Как запустить

1. скомпилировать проект самостоятельно:
   - Установить Python 3.11 или выше
   - Установить зависимости:
     ```bash
     pip install pyqt6 pymysql
     ```
   - Создать `.exe` файл:
     ```bash
     python -m PyInstaller --onefile --windowed --hidden-import=pymysql main.py

     ```
